## Procedural Character Recoil Animation

These scripts allow you to add dynamic, procedural animations to a selected bone when firing a weapon (spine bone is recommended)

## Usage

- Add the **CharacterProceduralRecoil** script to the character you want to use it on
- Add the **WeaponProceduralRecoil** script to the weapon to configure the recoil settings
